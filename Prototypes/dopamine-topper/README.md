# Dopamine Topper Prototype

A lightweight prototype exploring a feed-based social media experience where each item is a quiz from a Quiz Master.

## Features

- Interactive quiz posts authored by a Quiz Master.
- Engaging discussion section for each post, storing comments locally.
- Ability to filter the feed by locking onto the current post's `POST_TYPE` or `POST_GENRE`.

Open `index.html` in a browser to try it out.
